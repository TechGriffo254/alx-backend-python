# Python Decorators Project

This project demonstrates the use of Python decorators for database operations.

## Setup

1. Run the database setup script:
```bash
python setup_database.py
```

2. Run individual task files:
```bash
python 0-log_queries.py
python 1-with_db_connection.py
python 2-transactional.py
python 3-retry_on_failure.py
python 4-cache_query.py
```

## Tasks

- **Task 0**: Logging Database Queries
- **Task 1**: Handle Database Connections with a Decorator
- **Task 2**: Transaction Management Decorator
- **Task 3**: Retry Database Queries
- **Task 4**: Cache Database Queries
